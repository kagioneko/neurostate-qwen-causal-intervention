## v0.20 Result

keeping target layer 13 fixed.

Main result:

- `token_position = -1`

  - sign-flip p: `0.001953`
  - random residual controls with absolute slope >= residual: `0/15`
- `token_position = -2`
  - residual slope: `+0.004859`
  - random abs >= residual: `8/15`
- `token_position = -4`
  - residual slope: `-0.000846`
  - random abs >= residual: `13/15`
- `token_position = -8`
  - residual slope: `+0.015591`
  - random abs >= residual: `5/15`

Interpretation:

The layer-13 NeuroState residual effect is strongest at the immediate response
boundary (`-1`). Farther back in the prompt, the effect is much weaker and does
not clearly separate from random residual-space controls.
